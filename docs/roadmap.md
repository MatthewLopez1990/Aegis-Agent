# Roadmap

## Next Product Hardening

- Add live external database migration adapters beyond private operator-runner bundles for PostgreSQL/MySQL.
- Add richer policy rollout orchestration beyond due approved activation workers and multi-environment promotion artifacts.
- Enable more live channel adapters beyond signed webhook, brokered chat webhooks, and approval-gated SMTP email.
- Add more live implementations for selected high-value tools beyond allowlisted HTTP reads, approved HTTPS REST/webhook writes, live web search, live weather, and live geocoding.
- Add richer web run streaming controls beyond authenticated cursor reconnects, task pause/cancellation, per-step progress grouping, aggregate live progress metrics, and sanitized provider/tool substep telemetry.
- Add richer memory recertification workflows beyond dry-run previews, review digests, overdue escalation summaries, scheduled operator digest/escalation delivery, configurable per-type stale-confirmation marking, and per-team escalation route policies.
- Add additional exact tokenizer integrations beyond optional `tiktoken` and local SentencePiece context accounting.
- Add stronger OS-level isolated process profiles beyond no-network, ephemeral Python subprocess adapters, and CPU/address-space resource limits for generated skills.
- Expand self-repair beyond preflighted and synthesized patch candidates into fully live model-generated patch proposals with stronger isolated verification loops.

## Connector Growth

- OAuth-backed Microsoft Graph and Google Workspace flows plus provider-specific mail/calendar/contact rollback beyond allowlisted JSON reads and brokered-token productivity writes.
- GitLab connector and authenticated GitLab write adapters with scoped modes.
- Provider-specific Jira/Linear and ServiceNow field mapping, OAuth flows, and rollback adapters beyond the current brokered-token HTTPS write path.
- Provider-specific SIEM streaming adapters beyond local redacted JSONL audit export.

## Evaluation Growth

- Expand the initial prompt injection, memory poisoning, connector abuse, and skill escalation scenario corpus with multi-step attack chains and connector-specific variants.
- Add richer policy regression variants beyond the current high-risk policy class gate manifest, admin-only profile variants, bundle rollout canaries, malformed-receipt canaries, rollback canaries, policy bundle diff fuzz cases, CLI/API parity checks, TUI/web UI workflow parity canaries, persisted evaluation run reports with trend summaries, scheduled evaluation runs with operator review digests, scheduled scenario suites with reviewer assignment queues, reviewer disposition workflows, trend dashboards, automatic regression deltas between evaluation runs, policy promotion release gates, and release readiness summaries for operators.
